Torogo

Torogo is a free Android app to play regular and toroidal go over the internet

Download this project as a .zip file Download this project as a tar.gz file

A new way to Go

Toroidal go is an elegant variant of the game of go. It requires no special corner and border tactics (Joseki). The board scrolls and wraps around to reveal only middle space.

How to play

Install the Sneer chat app. Install the Torogo sneer app. Send an invite to a friend. While chatting to your friend, tap on the "apps" button. Search for Apps > Torogo. Play!

Authors and Contributors
